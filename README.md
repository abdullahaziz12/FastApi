# 🚀 FastAPI Practice Lab

A growing collection of hands-on FastAPI scripts as I learn the framework — part of my [self-directed backend development roadmap](https://github.com/abdullahaziz12). Every file here is written, tested, and pushed as I cover a new concept, so this repo doubles as a **learning log** and a **quick-reference cheat sheet**.

> 📌 **Status:** Actively updated. New files are added as I progress through the roadmap.

---

## 🧠 About This Repo

This repository is a practical playground for learning **FastAPI** — one of the fastest and most modern Python web frameworks. Instead of scattering practice code across random folders, everything lives here, organized and documented, so I (and anyone else learning FastAPI) can revisit concepts easily.

Each script is self-contained and focuses on **one core concept at a time** — routing, validation, CRUD operations, request/response models, and more.

---

## 📂 Repository Structure

| File | Concept Covered |
|---|---|
| `GetRoutes.py` | Basics of FastAPI — creating the app instance and simple `GET` routes |
| `QueryParms.py` | Handling **query parameters** (optional, required, defaults) |
| `Dynamic_Routing&Type_Validation.py` | **Path parameters**, dynamic routing, and automatic type validation |
| `Pydantic_Model.py` | Defining **Pydantic models** for request/response data validation |
| `PostApi.py` | Building **POST** endpoints to accept and process incoming data |
| `Calculator.py` | A simple calculator API — combining routes, params, and logic |
| `Crud_Operation.py` | Full **CRUD** (Create, Read, Update, Delete) operations — a mini Todo-style API |

> 📁 As new topics (databases, authentication, dependency injection, middleware, etc.) are covered, new files will be added and this table updated.

## 🛠️ Tech Stack

- **Language:** Python 3
- **Framework:** [FastAPI](https://fastapi.tiangolo.com/)
- **Server:** [Uvicorn](https://www.uvicorn.org/) (ASGI server)
- **Validation:** [Pydantic](https://docs.pydantic.dev/)
- **Docs:** Swagger UI (`/docs`) & ReDoc (`/redoc`) — auto-generated by FastAPI

## ⚙️ Getting Started

Want to run any of these scripts locally? Follow these steps:

### 1. Clone the repository
```bash
git clone https://github.com/abdullahaziz12/<repo-name>.git
cd <repo-name>
```

### 2. Create & activate a virtual environment
```bash
python -m venv venv
venv\Scripts\activate      # Windows
source venv/bin/activate   # macOS/Linux
```

### 3. Install dependencies
```bash
pip install fastapi uvicorn
```

### 4. Run any file
```bash
uvicorn <filename_without_extension>:app --reload
```
Example:
```bash
uvicorn GetRoutes:app --reload
```

### 5. Explore the interactive docs
Once running, open your browser at:
- Swagger UI → `http://127.0.0.1:8000/docs`
- ReDoc → `http://127.0.0.1:8000/redoc`
